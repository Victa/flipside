# Flip Side

A native iOS app for scanning and identifying vinyl records using computer vision and the Discogs database.

## Project Status

**Milestone 1: Image Capture Foundation** - ✅ Task 1 Complete

### Completed
- ✅ Xcode project initialized with SwiftUI + SwiftData (iOS 17.0 target)
- ✅ Camera and Photo Library usage descriptions added to Info.plist
- ✅ SwiftData models defined (Scan, ExtractedData, DiscogsMatch, SocialContext)
- ✅ Basic TabView navigation structure (Scan tab + History tab)
- ✅ Project configured for iPhone-only deployment

## Project Structure

```
FlipSide/
├── FlipSide.xcodeproj/          # Xcode project (generated by xcodegen)
├── FlipSide/
│   ├── FlipSideApp.swift        # Main app entry point with SwiftData
│   ├── ContentView.swift        # TabView with Scan and History tabs
│   ├── Info.plist              # App configuration with camera/photo permissions
│   └── Models/
│       └── Scan.swift          # SwiftData models
├── project.yml                  # XcodeGen configuration
└── Plan.md                      # Detailed architecture and milestone plan
```

## Requirements

- iOS 17.0 or later
- Xcode 15.0 or later
- Swift 5.9

## Opening the Project

Open `FlipSide.xcodeproj` in Xcode to build and run the app.

## Next Steps

See `Plan.md` for the complete milestone-based development plan. Next up:
- Milestone 1 (remaining tasks): ImageCaptureService, KeychainService, SettingsView
- Milestone 2: Vision Integration with OpenAI GPT-4o-mini
- Milestone 3: Discogs API integration

## Technology Stack

- **UI Framework**: SwiftUI (native iOS)
- **Persistence**: SwiftData
- **Camera**: AVFoundation + PHPickerViewController
- **Vision API**: OpenAI GPT-4o-mini
- **Database**: Discogs API
- **Security**: iOS Keychain for API keys
